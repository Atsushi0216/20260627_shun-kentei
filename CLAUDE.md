## V3 → 食/旬検定 の変更マップ

### 差し替える（＝唯一の実装作業）

6. **テーマ色・タイトル・コピー文言** ← 食/旬のトーンに
7. **OGP画像・favicon** ← 食テーマに

### そのまま流用する（触らない）

## 問題JSONスキーマ（食/旬検定版）

```json
{
  "id": "shun-001",
  "domain": "season | megiki | hozon | tabegoro | hinshu | tsukaikiri",
  "category": "🍑 桃",
  "text": "○○は△△である。",
  "correct": "good | bad",
  "explanation": "農家アツシの解説（1〜3文）",
  "difficulty": 1
}
```

- V3 の `munakata_step` → `domain` にリネーム（食の6ドメインキー）。
- V3 の `profession`・`layer` → 不要（1プール制）。
- `pickQuestions()` は `q.munakata_step` → `q.domain` に読み替えるだけ。
